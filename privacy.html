Politica de Privacidade

Ultima atualizacao: 17 de abril de 2026

1. Introducao

O EuAds e uma ferramenta de gerenciamento de anuncios para Meta Ads. Esta politica descreve como coletamos, usamos e protegemos suas informacoes ao utilizar nosso aplicativo.

2. Dados que coletamos

Para funcionar corretamente, o EuAds acessa os seguintes dados atraves da API do Meta:

  • Informacoes de contas de anuncios (ID, nome, status)
  • Dados de campanhas, conjuntos de anuncios e anuncios
  • Paginas do Facebook vinculadas a conta
  • Tokens de acesso fornecidos voluntariamente pelo usuario

3. Como usamos os dados

Os dados sao utilizados exclusivamente para:

  • Criar, editar e gerenciar campanhas de anuncios na plataforma Meta
  • Exibir metricas e status das campanhas ao usuario
  • Agendar e automatizar a publicacao de anuncios

4. Armazenamento

Todos os dados sao armazenados localmente no dispositivo do usuario. O EuAds nao envia, compartilha ou armazena dados em servidores externos. Tokens de acesso sao salvos localmente em arquivos de configuracao protegidos.

5. Compartilhamento de dados

O EuAds nao vende, compartilha ou transfere seus dados para terceiros. A unica comunicacao externa e feita diretamente com a API oficial do Meta (Facebook/Instagram) para executar as operacoes solicitadas pelo usuario.

6. Seguranca

Adotamos medidas para proteger seus dados, incluindo:

  • Armazenamento exclusivamente local
  • Tokens salvos com tipo password (mascarados na interface)
  • Conversao automatica para tokens de longa duracao (long-lived)

7. Seus direitos

Voce pode a qualquer momento:

  • Revogar o acesso do app nas configuracoes do Facebook
  • Excluir todos os dados locais removendo os arquivos de configuracao
  • Solicitar informacoes sobre quais dados estao armazenados

8. Exclusao de dados

Para excluir todos os seus dados do EuAds, basta remover a pasta de configuracao do aplicativo no seu dispositivo. Como nenhum dado e armazenado externamente, a exclusao local e completa e definitiva.

9. Contato

Para duvidas sobre esta politica de privacidade, entre em contato pelo repositorio do projeto ou diretamente com o administrador do aplicativo.
